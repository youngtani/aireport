## AI 수업 리포트 생성기 (HTML/CSS/JS)

튜터가 **학생 이름 / 오늘 읽은 책 / 클릭한 포인트(자주 쓰는 문구)**만 입력하면, 학부모에게 보낼 **한국어 수업 리포트 메시지**를 자동으로 생성합니다.

## 실행 방법 (Windows PowerShell)

### 1) 설치

프로젝트 루트에서:

```bash
npm install
```

### 2) OpenAI API Key 설정

#### 옵션 A) (추천) 현재 PowerShell 세션에만 설정

프로젝트 루트에서:

```bash
$env:OPENAI_API_KEY="YOUR_KEY_HERE"
npm start
```

> 이 방법은 **현재 터미널 창**에만 적용됩니다. 새 터미널을 열면 다시 설정해야 합니다.

#### 옵션 B) `.env` 파일로 설정 (로컬 파일)

프로젝트 루트에 `.env` 파일을 직접 만들어 아래처럼 넣어주세요:

```env
OPENAI_API_KEY=YOUR_KEY_HERE
PORT=3000
```

> 주의: `.env`는 커밋하지 않는 것을 권장합니다.

### 3) 실행

```bash
npm start
```

브라우저에서 `http://localhost:3000` 을 여세요.

## 사용법

- **학생 선택**: 우측의 “목록 관리”에서 학생을 추가/삭제할 수 있습니다(브라우저 로컬 저장).
- **책 제목 입력**
- **포인트 클릭**: 자주 쓰는 문구를 눌러 포함할 내용을 선택
- **Make report**: 결과 생성 → **복사** 버튼으로 바로 복사
- **최근 생성 기록**: 다시 불러오거나 복사 가능(최대 20개 저장)

## 모델

서버(`server.js`)는 OpenAI Responses API를 통해 `gpt-5` 모델로 생성합니다.
